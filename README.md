## Support for

### DSL Queries

#### Compound queries
* __bool__
* __boosting__

* __match_phrase_prefix__
* __match_phrase__
* __combined_fields__
* ~~__query_string__~~

#### Geo queries
* [__geo_bounding_box__](https://www.elastic.co/guide/en/elasticsearch/reference/current/query-dsl-geo-bounding-box-query.html)
* __geo_distance__
